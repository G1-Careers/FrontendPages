Training Curriculum

Our 6-week Data Science & Machine Learning program equips you with the skills to excel in the industry:


  • Week 1: Foundations + Python for ML - Understand AI vs ML with real-world examples, learn Python essentials (variables, functions, loops, lists), and work with NumPy & Pandas for efficient data handling.
  • Week 2: Math, Probability & ML Basics - Refresh key concepts in linear algebra and probability, dive into Supervised Learning fundamentals, and build your first Linear & Logistic Regression models.
  • Week 3: Core Machine Learning Algorithms - Learn key ML algorithms (Decision Trees, k-NN), evaluate models using accuracy, precision, recall, F1-score, and complete Industry Project 1: Apply ML techniques to real-world datasets for prediction & classification.
  • Week 4: Deep Learning Foundations - Explore Neural Networks (perceptrons, activation functions), build your first deep learning model using TensorFlow/Keras, and understand overfitting, dropout, and regularization. Intro to CNNs and their role in image recognition.
  • Week 5: CNN Project + NLP Basics - Complete Industry Project 2: Implement a CNN for image-based classification tasks, introduction to Natural Language Processing, text preprocessing (tokenization, stopword removal), build a text classifier using bag-of-words & TF-IDF, visualize word embeddings, and explore ML model deployment using Streamlit.
  • Week 6: Capstone Project & Demo - Capstone Industry Project: Apply everything learned to solve a real-world AI-ML problem, work on model building, tuning, and deployment, and present your project with code, results, and visualizations.
